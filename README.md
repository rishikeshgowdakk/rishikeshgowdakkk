# HTML, CSS, jQuery Portfolio

This is a simple one-page portfolio website built using only HTML, CSS, and JavaScript with the jQuery library. It was created to fulfill a specific university assignment requirement.

## How to Run

1. Make sure you have Node.js and npm installed.
2. Open a terminal in the project directory.
3. Run `npm install` to install the local HTTP server.
4. Run `npm start`.
5. Open your browser and go to `http://localhost:9002`.
